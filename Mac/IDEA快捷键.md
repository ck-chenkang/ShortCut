# IDEA常用快捷键  - Mac

| 命令                                    | 快捷键              | 说明 |
| --------------------------------------- | ------------------- | ---- |
| 切换主题、大小等                        | Ctr + ~             |      |
| 设置                                    | Cmd + ,             |      |
| 在上方插入行                            | Opt + Cmd + Enter   |      |
| 在下方插入行                            | Shift + Enter       |      |
| 快速生成get、 set、 构造函数、 tostring | Cmd + N             |      |
| 智能补齐                                | Opt + Enter         |      |
| 快速启动当前函数                        | Shift + Control + R |      |
| 快速打开                                | 连按shift两次       |      |
|                                         |                     |      |
